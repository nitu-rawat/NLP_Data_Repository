# NLP_Data_Repository
Incident Classification System

Overview

This project is a simple Python-based Incident Classification System that automatically assigns IT support incidents to the appropriate support group based on keywords found in the incident description.

The system helps streamline the ticket routing process by reducing manual effort and ensuring incidents are directed to the correct team.

Features

- Automatic incident categorization
- Keyword-based classification
- Easy to understand and modify
- Suitable for learning basic IT Service Management (ITSM) concepts

Technologies Used

- Python 3

Support Groups

Keyword Category| Assigned Team
Internet, WiFi| Network Support Group
Server, Database| Server Database Team
Password, Login| Service Desk (L1 Support)
Application, Software| Application Support Team
Security, Virus| Security Team
Others| General IT Support

Sample Input

- Internet connection is very slow
- Unable to login due to issue
- Application is crashing frequently
- Server database not responding

Sample Output

- Network Support Group
- Service Desk (L1 Support)
- Application Support Team
- Server Database Team
